# 📁 Data Directory

This directory contains SLA contracts and processed outputs used by SentryChain AI.

> ⚠️ **Raw contracts are not included in this repository** due to file size and confidentiality. You must add your own contracts to run the pipeline.

---

## 📂 Directory Structure

```
data/
├── contracts/                        # Place your SLA PDF files here
│   ├── your_contract_1.pdf
│   └── your_contract_2.pdf
│
└── processed_contracts/              # Auto-generated by the pipeline
    ├── your_contract_1_parsed.txt    # Extracted text (LlamaParse)
    ├── your_contract_1.json          # Structured metadata (LlamaCloud)
    ├── your_contract_2_parsed.txt
    └── your_contract_2.json
```

## 🚀 How to Add Your Own Contracts

1. Place your PDF files in `data/contracts/`
2. Run the ingestion pipeline:
   ```bash
   python -m src.main  # with RUN_INGESTION = True
   ```
3. Processed `.txt` and `.json` files will be auto-generated in `data/processed_contracts/`
